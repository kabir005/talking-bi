## 🛠️ TECHNICAL STACK

### Backend
- **Runtime:** Python 3.11+
- **Framework:** FastAPI (async)
- **Database:** SQLite + SQLAlchemy
- **Data:** Pandas, NumPy
- **ML:** scikit-learn, PyCaret
- **Memory:** FAISS, ChromaDB
- **Graphs:** NetworkX
- **Scraping:** Playwright
- **Reports:** ReportLab, python-pptx
- **LLM:** Groq API (free tier)

### Frontend
- **Framework:** React 18 + TypeScript
- **Build:** Vite
- **Styling:** TailwindCSS
- **Charts:** Recharts, Chart.js
- **Layout:** react-grid-layout
- **Maps:** Leaflet
- **Graphs:** react-force-graph
- **State:** Zustand
- **HTTP:** Axios

## 🚀 DEPLOYMENT READINESS

### Backend Deployment
- **Platform:** Render.com (free tier)
- **Requirements:** Python 3.11+, 512MB RAM
- **Environment:** Set GROQ_API_KEY
- **Database:** SQLite (included)
- **Status:** Ready to deploy

### Frontend Deployment
- **Platform:** Vercel (free tier)
- **Build:** `npm run build`
- **Environment:** Set API_URL
- **Status:** Ready to deploy

### Quick Start
```bash
# Backend
cd talking-bi/backend
pip install -r requirements.txt
python main.py

# Frontend
cd talking-bi/frontend
npm install
npm run dev
```
